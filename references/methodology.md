# 沪深300对冲基差方法说明

## 默认数据源

- 沪深300现货：Sina 指数快照接口 `sh000300`
- IF 合约：Sina 期货快照接口 `nf_IFYYMM`
- 合约月份规则：按中金所沪深300股指期货产品规则处理

中金所产品页说明 IF 合约月份为“当月、下月及随后两个季月”，最后交易日为“合约到期月份的第三个周五，遇国家法定假日顺延”。这个 skill 用这个规则推导活跃合约，但为了轻量和稳健，不直接依赖中金所页面抓表。

## 公式

定义：

- `S` = 沪深300现货点位
- `F` = IF 合约最新价
- `D` = 距离到期的自然日天数

计算：

- 基差点数：`basis_points = F - S`
- 基差率：`basis_rate = (F / S - 1) * 100%`
- 年化基差率：`annualized_basis = (F / S - 1) * 365 / D * 100%`

解释：

- `basis_points < 0`：贴水
- `basis_points > 0`：升水

## 对中性对冲产品的含义

这里默认产品近似为：

- 多头：现货股票篮子、指数增强组合或 ETF 替代
- 空头：IF 对冲

在这个框架下：

- IF 贴水越深，空头开仓价格越低，未来向现货收敛时，通常意味着负 carry 更重
- IF 升水越高，空头开仓价格越高，未来向现货收敛时，通常意味着正 carry 更好

这是粗口径，不等于产品真实净值表现。真实结果还会被以下因素显著影响：

- 组合相对沪深300的暴露偏差
- Alpha 与选股收益
- 分红点位与股息调整
- 换月执行价差
- 交易费用和冲击成本

## 主力合约判定

优先使用 Sina 期货快照里的主力标记字段；如果字段异常，再回退到活跃合约中持仓量最大的 IF 合约。

## 到期日与剩余天数

脚本默认把每个合约的最后交易日近似为该月第三个周五。

注意：

- 中金所规则里，遇国家法定假日要顺延到下一交易日
- 这个 skill 没有内置中国节假日日历，因此把到期日作为“粗看 carry”的近似值
- 如果用户要求精确到交割日、换月日或节假日顺延，请先做人工核对

## Computer Use Fallback

如果遇到以下情况，可以改走浏览器人工校对：

- Sina 接口返回空字符串
- 返回时间戳明显陈旧
- SSL 或网络环境导致接口反复失败
- 用户要求截图式、页面式确认

建议用 `Computer Use` 打开并核对：

- 中金所沪深300股指期货产品页：`https://www.cffex.com.cn/hs300/`
- 中金所延时行情页：`https://www.cffex.com.cn/yshq/`

人工校对时仍然优先核对三件事：

- 主力合约是哪一个
- 主力与近月是升水还是贴水
- 年化基差大概落在哪个区间
